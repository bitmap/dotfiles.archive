# .dotfiles

```sh
git clone git@github.com:bitmap/.dotfiles.git && cd .dotfiles
```

## Install

Switch your shell to `zsh` to insure all scripts work correctly

```sh
chsh -s $(which zsh)
```

and run the install script

```sh
cd .dotfiles
./install
```

## Run install script for macOS

```sh
cd .dotfiles
./mac/install
```
